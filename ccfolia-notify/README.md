# 🔔 ココフォリア チャット通知

ココフォリア（[ccfolia.com](https://ccfolia.com/)）のチャットに新しいメッセージが投稿されたとき、通知音を鳴らすChrome拡張機能です。

別タブを見ている間や、画面から目を離しているときでも、発言に気づくことができます。

---

## 動作確認済み環境

- Google Chrome（最新版）
- OS：macOS

※ Firefox・Edgeなど他のブラウザには対応していません。

---

## インストール方法

Chromeウェブストアには公開していません。以下の手順で手動インストールしてください。

**1. ファイルをダウンロードする**

このページの右上にある `Code` ボタン →「Download ZIP」を選択し、ZIPファイルをダウンロードします。ダウンロードしたZIPを展開（解凍）してください。

**2. Chromeの拡張機能管理ページを開く**

アドレスバーに以下を入力してEnterを押します。

```
chrome://extensions/
```

**3. デベロッパーモードをオンにする**

ページ右上の「デベロッパーモード」のトグルをオンにします。

**4. 拡張機能を読み込む**

「パッケージ化されていない拡張機能を読み込む」ボタンが表示されるのでクリックし、先ほど展開したフォルダを選択します。

**5. 完了**

拡張機能一覧に「ココフォリア チャット通知」が表示されれば完了です。ココフォリアを開いてチャットが届いたとき、通知音が鳴ります。

---

## 使い方・設定

Chromeのアドレスバー右側にある🧩（パズルピース）アイコンをクリックすると、インストール済みの拡張機能一覧が表示されます。「ココフォリア チャット通知」の右にある📌（ピン）アイコンを押すと、ツールバーに常時表示されて便利です。

ツールバーに表示された拡張機能アイコンをクリックすると、設定ポップアップが開きます。

### 通知を有効にする

チェックを外すと、通知音が完全にオフになります。一時的に静かにしたいときに使ってください。

### 通知対象タブ

通知音を鳴らすチャットタブを絞り込めます。

| 設定値 | 内容 |
|--------|------|
| メインのみ | メインタブの発言のみ通知 |
| メイン＋雑談 | メインと雑談タブのみ通知 |
| 全タブ | すべてのタブを通知（初期値） |

### 通知音

4種類から選べます。「試聴」ボタンで音を確認できます。

| 選択肢 | 内容 |
|--------|------|
| ピッ（短音） | シンプルな単音 |
| ピピッ（二連音） | 短い2連音 |
| ピ〜（上昇音） | 音程が上がるアラーム風 |
| チャイム | やさしいチャイム音 |

### 音量

スライダーで0〜100%の範囲で調整できます。

### 自分の発言は通知しない

オンにすると、自分のキャラクター名が含まれる発言では通知音が鳴りません。キャラクター名の入力欄に、ココフォリアで使っている名前を入力してください。

> ⚠️ キャラクター名は**部分一致**で判定されます。例えば「アリス」と設定すると「黒アリス」の発言も通知されません。

---

## 注意事項

- ココフォリアのUI構造が更新された場合、動作しなくなることがあります。
- 通知音はブラウザの音声APIを使用しています。ページを開いた直後（ユーザー操作前）は音が出ないことがあります。
- 本拡張機能はココフォリアのデータをサーバーに送信したり、外部と通信したりすることは一切ありません。
