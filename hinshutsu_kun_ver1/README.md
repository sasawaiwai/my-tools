# 頻出くん ver1

原稿校正補助ツール（頻出表現検出）

## 概要

頻出くんは、文章内で繰り返し使われている表現を機械的に検出する校正補助ツールです。
N-gram方式を使用し、「とにかく重複していれば拾う」ことを目的に設計されています。

- 意味や文脈の判断は行いません
- 文章の自動修正・自動改善は行いません
- 文章の傾向を可視化するための補助ツールです
- **完全ローカル動作**（入力テキストが外部に送信されることはありません）

## 使い方

1. `hinshutsu_kun_ver1.html` をブラウザで開く
2. 原稿テキストを貼り付ける（またはファイルを読み込む）
3. 「分析実行」ボタンを押す
4. 検出結果を確認する

### 推奨ブラウザ（Mac で動作確認済み）

- Chrome
- Safari

### 対応ファイル形式

- `.txt`
- `.docx`

## 機能

- **N-gram解析** による頻出表現の自動検出
- **ユーザー辞書** ：特にチェックしたい表現を登録できます
- **除外語** ：ノイズになりやすい語（助詞など）を検出結果から除外できます
