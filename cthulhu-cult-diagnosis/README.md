# クトゥルフ神話TRPG・狂信者適性診断

クトゥルフ神話TRPGをテーマにした、ジョーク診断サイトです。  
14の質問に答えると、あなたが最も「狂信者」として適性のある神格が診断されます。

## 概要

- 14問の設問に回答
- 16柱の旧支配者・神格の中から結果を判定
- 結果はクリップボードにコピー可能
- スマートフォン対応

## 注意事項

- 本サイトは**非公式ファンコンテンツ**です
- クトゥルフ神話TRPGの公式・権利者とは一切関係ありません
- **実在の宗教・団体・信仰とは関係ありません**
- ルールブック本文・既存書籍の文章の引用・転載はしていません
- あくまでジョーク診断としてお楽しみください
